# /convert [from] [to]

Convert between different formats.

## Description
Converts data between various formats like JSON to TypeScript, SQL to ORM models, or CSV to JSON.

## Usage
```
/convert json typescript
/convert openapi typescript
/convert sql prisma
/convert csv json
```

## Output Includes
- Converted output
- Type definitions
- Validation rules
- Helper functions
- Usage examples
- Edge case handling
- Format documentation
